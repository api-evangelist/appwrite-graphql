# Appwrite GraphQL Plans

Appwrite offers three plan tiers. The GraphQL API is available across all plans. Plans are applied at the organization level, with resources allocated per project.

## Free Plan

**Cost:** $0/month

- 5 GB bandwidth per project
- 2 GB storage per project
- 750,000 function executions/month
- 75,000 monthly active users
- 1 Database, 1 Bucket, 2 Functions per project
- 15-minute build timeouts
- Community support only
- Limit of 2 projects (shared resources)
- Projects pause after 1 week of inactivity

## Pro Plan

**Cost:** Starting at $25/month (most popular)

- 2 TB bandwidth per project
- 150 GB storage per project
- 3,500,000 function executions/month
- 200,000 monthly active users
- Unlimited Databases, Buckets, and Functions
- 45-minute build timeouts
- Email support with daily backups (7-day retention)
- Organization roles and add-ons available
- Budget caps and alerts
- Dedicated resources per project

**Overage charges:**
- Additional bandwidth: $15 per 100 GB
- Additional storage: $2.80 per 100 GB
- Database reads: $0.06 per 100,000
- Database writes: $0.10 per 100,000

## Enterprise Plan

**Cost:** Custom pricing

Includes all Pro features plus:
- Uptime SLAs and designated success manager
- Up to 24/7 support with private Slack channel
- SOC-2, HIPAA, and BAA compliance
- Custom organization roles, SSO, and activity logs
- Advanced observability and log drains (90-day retention)
- Bring your own cloud option

## Self-Hosted

Appwrite is open source and can be self-hosted at no software cost. The GraphQL API is fully available in self-hosted deployments.
